安装

首先要准备一个安装介质。可以下载一个系统镜像 iso 文件， 刻到 CD 或者 DVD 或者 U 盘上。高级的安装方式还有硬盘安装、 网络安装等。个人推荐下载 600 MB 左右大小的 LiveCD 镜像文件， 刻到 U 盘中，制作成一个可启动的 Live USB。Live USB 制作工具一般 用 UNetBootin，Fedora 系统的话可以用软件源中的 LiveUSB Creator 制作。我手头有一个 Fedora 14 的 Live USB，还有一个 Ubuntu 10.04 的 CD 光盘。

如果打算安装双系统（如 Windows+Linux）， 推荐在安装前事先预留出一块硬盘空间用来安装 Linux。方法时 用磁盘管理工具或专门的分区工具删除/缩小一个现有分区（如H盘）。 如果打算用虚拟机软件安装 Linux，只需在虚拟机中建立虚拟硬盘 即可，通常在外边系统上就是一个文件，在虚拟机中的系统看来是 一块完整硬盘。

安装过程已经很简单易用了。只需配置一下时区（城市选上海）、 root （管理员）的密码等。提醒硬盘分区时注意小心，不要把有 重要数据/需保留数据的分区误删或误格式化。

安装完成后，重启后第一次进入 Linux 系统，可能需要做更多的 配置，如新建一个普通用户。
