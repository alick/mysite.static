GPRS overview

From GPRS on wikipedia:

  • now maintained by 3GPP
  • best-effort, variable throughput and latency
  • moderate-speed data transfer using unused TDMA channels
  • IP packet: IPv4
  • Hardware: Class B(GPRS suspended--resumed)
  • Addressing: reference to APN. APN defines the services(WAP/SMS/WWW)
  • Multiple access method: FDD and TDMA
  • packet: constant length, GSM time slot
  • down-link: first-come first-serve
  • up-link: Reservation ALOHA
  • channel encoding: convolutional code, GMSK modulation
  • shared nature, contrast to HSCSD
  • multislot class: high speed
  • low speed, long RTT
  • prioritized lower than speech

No. 7信令

特点

  • 用于交换机间、交换机和数据库(如HLR、VLR和AUC)之间
  • 公共信道信令(CCS)：信令信号和业务信号分离
  • 灵活、速度快、易扩充

协议组成

  1. MTP 消息传递部分。提供无连接的消息传输。分三层：
    1. 信令数据层。信号信道的物理和电气特性。OSI 物理层
    2. 信令链路层。可靠的数据传输。OSI 数据链路层
    3. 信令网络层。公共的消息传送功能。OSI 网络层
  2. SCCP 信令连接控制部分。提供数据的无连接和面向连接业务。
    • 无连接：移动用户的鉴权、智能用户的帐号查询
    • 有连接：
  3. TUP 电话用户部分。电话呼叫有关的信令
  4. ISUP ISDN 用户部分
  5. TACP 事务处理应用部分。应用层。为各种业务提供接口。
  6. MAP 移动应用部分。移动网网内和网间互连

参见： Signaling System No. 7 (SS7/C7): Protocol, Architecture, and Services

Control Channels

  • BCCH: Broadcast control channel 广播
    • BCCH 广播控制信道
    • FCCH 频率修正信道
    • SCH 同步信道
  • CCCH 公共控制信道
    • PCH(Paging channel) 寻呼信道 only downlink, handling incoming call
    • RACH 随机接入信道 only uplink, requesting connection establishment
    • AGCH 接入确认信道 BTS inform MS outcome of connection setup request
  • DCCH
    • SDCCH 独立专用控制信道
    • SACCH 慢相关控制信道
    • FACCH 快相关控制信道
