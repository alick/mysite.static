GPRS overview

From GPRS on wikipedia:

  • now maintained by 3GPP
  • best-effort, variable throughput and latency
  • moderate-speed data transfer using unused TDMA channels
  • IP packet: IPv4
  • Hardware: Class B(GPRS suspended--resumed)
  • Addressing: reference to APN. APN defines the services(WAP/SMS/WWW)
  • Multiple access method: FDD and TDMA
  • packet: constant length, GSM time slot
  • down-link: first-come first-serve
  • up-link: Reservation ALOHA
  • channel encoding: convolutional code, GMSK modulation
  • shared nature, contrast to HSCSD
  • multislot class: high speed
  • low speed, long RTT
  • prioritized lower than speech

No. 7信令

特点

  • 用于交换机间、交换机和数据库(如HLR、VLR和AUC)之间
  • 公共信道信令(CCS)：信令信号和业务信号分离
  • 灵活、速度快、易扩充

协议组成

  1. MTP 消息传递部分。提供无连接的消息传输。分三层：
    1. 信令数据层。信号信道的物理和电气特性。OSI 物理层
    2. 信令链路层。可靠的数据传输。OSI 数据链路层
    3. 信令网络层。公共的消息传送功能。OSI 网络层
  2. SCCP 信令连接控制部分。提供数据的无连接和面向连接业务。
    • 无连接：移动用户的鉴权、智能用户的帐号查询
    • 有连接：
  3. TUP 电话用户部分。电话呼叫有关的信令
  4. ISUP ISDN 用户部分
  5. TACP 事务处理应用部分。应用层。为各种业务提供接口。
  6. MAP 移动应用部分。移动网网内和网间互连

参见： Signaling System No. 7 (SS7/C7): Protocol, Architecture, and Services

GSM TDMA Frame Structure

  • 每帧 4.615ms
  • TDMA。8 时槽/时隙，每时槽 0.557ms。
  • 帧长 156.25bit = 148 + 8.25 bit
  • modulation scheme in a slot: GMSK
  • 组合成复帧、超帧和超高帧
  • 发送/接收时槽有时间偏移

Logical Channels(Traffic Channels and Control Channels)

信道(channel) 映射为一个时隙的重复出现，对应一个频率和TDMA中的一个时槽。

Traffic Channels(TCH): speech and data

  • TCH/F: full rate channels
    • 26-Multiframe(120ms): 24 for traffic + 1 for SACCH + 1 unused
    • uplink/downlink separated by 3 bursts
  • TCH/H: half rate channels
    • double capacity
    • 26-Multiframe, internal different...

Control Channels: network management, channel maintainance

  • BCH(Broadcast channels): BS 广播给 MS
    • BCCH 广播控制信道 parameters needed to identify and access the network
    • FCCH 频率修正信道 frequency reference
    • SCH 同步信道 training symbol sequece
  • CCCH(Common Control Channels) 公共控制信道 establish the calls
    • PCH(Paging channel) 寻呼信道 only downlink, alert MS of incoming call
    • RACH(Random Access Channel) 随机接入信道 only uplink, requesting connection establishment
    • AGCH(Access Grant Channel) 接入确认信道 BTS inform MS outcome of connection setup request
  • DCCH(Dedicated Control Channels) message exchange
    • SDCCH(Standalone Dedicated Control Channel) 独立专用控制信道 exchange sigaling uplink and downlink
    • SACCH 慢相关控制信道 channel maintainance and control
    • FACCH 快相关控制信道 replace traffic channel when urgent sigaling needed
