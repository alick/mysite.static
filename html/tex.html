简介

TeX是一个专业的文档排版工具，用于生成高质量的文档书籍。如今 也被用于生成电子文档。TeX尤其擅长于对数学公式的排版。

TeX 现在已经被Knuth 冻结(frozen)，它的版本号也很有趣，每一个 修订版都将版本号增加一位，是收敛于π的。现在的最新版为3.1415926 。

这里有对TeX的历史、未来、优点和 不足等的中文简介。

LaTeX 是对 TeX 的扩展，它引入了文档类的概念， 并定义了许多高层的命令，使得普通人也可以很方便地写出高质量文章、 论文、书籍。

TeX 等名字会被它的排版引擎做特殊的处理。 使用 \TeX 可以得到 TeX， 使用 \LaTeX 可以得到 LaTeX， 使用 \XeTeX 可以得到 X∃TeX。

CTAN(Comprehensive TeX Archive Network)是提供最齐全的及时更新的与TeX相关的软件的一个站点，在 全球有许多镜像站点。中国大陆有ctex、厦门大学、 清华大学等镜像。

安装

详细的安装步骤可以参考这里的幻灯片。

下面是一个简要介绍：

要在 Linux 系统下安装 TeX 工具，推荐用网络安装的方式下载安装 TeX Live 最新版。 这需要首先下载一个 安装脚本， 名字为 install-tl-unx.tar.gz 。解压后在解压得到的目录下使用类似下面的命令安装：

sudo ./install-tl --gui=perltk -repository='ftp://ftp.ctex.org/mirrors/CTAN/systems/texlive/tlnet/'

也就是说，安装需要 root 权限（如果不选择安装到系统目录则不需要），指定使用 GUI 图形界面（需要系统上有 Perl 的 Tk 模块），指定从 ctex.org 的镜像下载需要安装的软件（各种宏包和文档）。
